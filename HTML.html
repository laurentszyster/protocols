HTML

The seven methods of HTML provide just enough support for the common DOM manipulations: replace an element, update an element content and insert HTML markup before or after the beginning or the end of an element; add and remove CSS classes; build a query string from HTML form input values; escape text as valid CDATA.

Many web applications can use these seven methods only, most entreprise applications will.

Fancy visual effects are expensive and tricky to run accross plateforms, drag and drop has allways been a gimmick with many pitfalls for its users and GUI widgets in JavaScript are web user interface non-senses. If you want the best web user interface possible everywhere, learn to apply HTML and CSS first: protocols.js is designed to leverage both.
