Synopsis

Easy things should be easy:

Hello !

Identify an element in the application's HTML page and attach it a the most common event handler written in the simplest and most readable way: 

<div class="test" onclick="JSON.GET('hello-world.js')" >
  Hello <span id="who" >...</span>
</div>

On click get the JSON resource named hello-world.js. 

{"who": "World"}

Update or extend the page with the response values

By convention, CSS class names and HTML id attributes are used throughout the library to link JSON namespaces to HTML templates and DOM elements. 

<div class="hidden" id="Protocols.JSON.templates">
  <span class="who"><em><json/><em></span>
</div>

Note how the source above acts as a substitute of traditionnal server-side templates and how it can be supported for architectural and demonstration purposes by a static JSON resource instead of an actual service controller (hello-world.js).
