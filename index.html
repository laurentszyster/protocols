Protocols is a small JavaScript library to support complex and large entreprise applications.

It combines the best of Jack Slocum's CSS and XPATH selector, bits of Douglas Crockford sagacity and pearls of common blog wisdom about incompatibilities between browsers.

With 45 symbols this library supports one practical way to apply HTTP, HTML, CSS and JSON in a few common browsers so that architects, analyst programmers and web designer may leverage together the stable stack of web protocols.
