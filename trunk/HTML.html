HTML

The ten methods of HTML provide just enough support for the common DOM manipulations: replace an element, update an element content and insert HTML markup before or after the beginning or the end of an element; add and remove CSS classes; attach event listeners; build a query string from HTML form input values; escape text as valid CDATA.

Many web applications can use these methods only, most entreprise applications will.

Fancy visual effects are expensive and tricky to run accross plateforms, drag and drop has allways been a gimmick with many pitfalls for its users and GUI widgets in JavaScript are web user interface non-senses. If you want the best web user interface possible everywhere then learn to apply HTML and CSS.

This library is designed to leverage both.

HTML.bind(element, type, function (event) {...})

Bind a function to an element as an listener to event of a given type. 

HTML.bind ($('html_bind_foobar'), 'click', function (event) {
    alert(this.innerHTML);
    return true;
    });
Click here to send an event to the attached listener.

Note how this in the bound function refers to the element it was binded to.
