Introduction

Protocols is a small JavaScript library to support complex and large entreprise applications.

It combines only the best of Jack Slocum's CSS and XPATH selector, some of Douglas Crockford sagacity and pearls of common blog wisdom about incompatibilities between browsers. But none of the fat I also found in Prototype, Mochikit, Dojo, jQuery and Ext.

With 43 symbols only, this library supports one practical way to apply HTTP, HTML, CSS and JSON in a few common browsers so that architects, analyst programmers and web designer may leverage together the stable stack of web protocols to integrate a growing number of web services into a single entreprise application's front-end.
